Simple encryption / decryption

Quickly and simply encrypt some text in the browser using open source. Feel free to inspect the code to see how it works, including the xxtea encryption algorithm.

Type in a password/passphrase, then paste in the text you want to encrypt. Then press the Encrypt button and copy the ciphertext. To decrypt, leave the plaintext field empty, enter the ciphertext with the same original password, and click Decrypt.

(NOTES: This uses Corrected Block TEA (often referred to as XXTEA), a simple to code cipher which is unencumbered by patents and can run on low power devices like Raspi. This entire website can be downloaded and run locally.)
